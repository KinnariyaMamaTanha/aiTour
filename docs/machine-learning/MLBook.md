# 机器学习书 MLBook

## 书籍简介

- **作者**：[KevinZonda](https://github.com/KevinZonda) 等
- **先修要求**：高中数学知识
- **主要内容**：机器学习中基础的数学和理论

这是 KevinZonda，一位英国伯明翰大学毕业生，基于伯明翰大学的几门课程搭建的机器学习书籍，聚焦于机器学习中基础的数学和理论。该书希望能满足*只有高中数学知识储备*的同学顺利打下机器学习基础。在我收录这本书时，该项目正在搭建中，可能存在错误和不完整的地方。

> Best Regards to KevinZonda.
