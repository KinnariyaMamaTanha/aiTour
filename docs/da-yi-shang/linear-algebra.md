# Linear Algebra

## 课程简介

* **先修要求**：无
* **学习材料**：
    * _Introduction to Linear Algebra_，作者：Gilbert Strang
    * 《高等代数学》，作者：姚慕生，吴泉水，谢启鸿
    * _Essence of Linear Algebra_，创作者：3blue1brown
* **主要内容**：
    * 线性方程组
    * 线性空间
    * 线性映射
    * 矩阵及行列式
    * 多项式
    * 特征值及对角化
    * 二次型
    * 内积空间
    * 奇异值分解
    * 快速傅里叶变换

## 个人心得

我认为，学习线性代数最重要的有两点：

1. 几何直观
2. 标准型思维
